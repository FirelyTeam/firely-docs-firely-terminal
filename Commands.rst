Commands
========

Firely Terminal has several groups of commands, for different topics: -
Packages - a FHIR client - Managing packages - REsolving canonical urls
- Managing a list of known FHIR servers - Connecting to Simplifier
(Projects) - FHIR resource manipulation, using a stack - Snapshot
Generation and Resource Validation
